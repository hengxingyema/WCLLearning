下面是服务器开发相关经历

  • Node.js Redis 解决同一秒内很多人抢任务的并发问题

    • 问题：服务器偶尔会有任务被超领的问题，下午2点-4点服务器速度会变慢
    • 确认代码逻辑是否正确，查看日志和服务器并发量
    • 进行高并发测试，从而确认是由于高并发访问引起
    • 调研解决高并发访问量的技术方案（淘宝）
      1. 采用Redis的技术方案来解决高并发
      2. 事务层面采用两段式方案（先申请，再确认）
      3. 利用redis事务操作的原子性，来确保高并发访问
      4. 把常用数据读入到redis中，并且保持和服务器数据库同步，提升服务器的整体访问速度

  • Node.js Express MongDB 云服务 公司网站产品的服务器
